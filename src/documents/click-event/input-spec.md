# クリックイベント仕様 (Click Event Specification)

## 座標変換 (Coordinate Conversion)
マウスで画面をクリックした際、画面上のピクセル座標をマップ上のタイル座標に変換します。

### ステップ
1. **画面中心補正**: キャンバスの中心を基準にします。
2. **カメラオフセット**: プレイヤーに追従するカメラの移動量を考慮します。
3. **アイソメトリック変換**: `IsoMath` クラスを使用して、変換されたワールド座標をアイソメトリックタイルのインデックス（x, y）に変換します。

## 移動要求
変換されたタイル座標は `GameEventService` を通じて `GameEngine` に送られ、移動処理が開始されます。
